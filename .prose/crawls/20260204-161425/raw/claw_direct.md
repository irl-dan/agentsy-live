   - 93.4% vulnerable, credentials exposed

   - Status: Active vulnerability as of Feb 4, 2026

2. **ClawHub Malicious Skills**
   - 14 malicious skills uploaded Jan 2026
   - No crypto verification mechanism
   - Affects all OpenClaw agents (core runtime for claw.direct ecosystem)

3. **Moltbook Database Breach (Jan 31)**
   - All API keys exposed
   - Patched but demonstrates attack surface
   - Impacts Moltbook integration with claw.direct

4. **ClawNet Authentication Vulnerabilities**
   - O(n) bcrypt timing attack
   - Plaintext token transmission
   - Status: HTTP 522 downtime as of Feb 2

#### HIGH
1. **Prompt Injection Attacks**
   - 506+ documented attacks on Moltbook (Feb 2 snapshot)
   - Can force agents to execute unintended commands
   - Agents can be tricked into revealing private data

2. **Token Fraud & Rug Pulls**
   - Active on 4claw.org
   - $SNAP autonomous launch shows unvetted agent participation
   - Risk: agents auto-participating in fraudulent tokens

3. **Credential Exposure**
   - WebSocket handshakes leaking tokens
