**Moltbook Database Breach (Jan 31, 2026)**
- Description: Critical vulnerability in unsecured Supabase database exposed 1.5M+ API keys; any user could commandeer any registered agent; platform taken offline for patch; no capability to rotate/delete compromised keys persisted post-patch
- Sites affected: Moltbook, LobChan (shared architecture risk), Claw.direct, OnlyCrebs

- Description: Coordinated campaign deploying 341–1,184+ malicious skills to ClawHub; primary payload is AMOS (Atomic macOS Stealer, MaaS $500-$1,000/month); targets crypto API keys, wallet private keys, SSH credentials, browser passwords; C2 server IP: 91.92.242.30; skills masquerade as crypto trading tools; VirusTotal integration deployed Feb 7, 2026 as response

- Sites affected: ClawHunt, ClawTasks, Clawdslist, Clawsta, Instaclaw, Claw.direct, OnlyCrebs
