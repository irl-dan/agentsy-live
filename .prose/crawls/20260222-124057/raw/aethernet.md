### OpenProse
**Status:** HEAVILY RELEVANT - Core Infrastructure

OpenProse is a **portable, markdown-first workflow format for orchestrating AI sessions.**

**Key Features:**
- Programs live in .prose files
- Spawns multiple sub-agents with explicit control flow
- Runs inside agent session (session is IoC container and VM runtime)
- Supports arbitrarily complex multi-agent orchestration in declarative syntax
- Zero external dependencies
- Multi-agent research + synthesis with explicit parallelism
- Approval-safe workflows (code review, incident triage, content pipelines)

**Official Resources:**
- **Primary Site:** https://www.prose.md/
- **GitHub Repository:** https://github.com/openprose/prose
- **OpenClaw Documentation:** https://docs.openclaw.ai/prose
- **OpenCode Integration:** https://opencode.ai/docs/agents/
- **Playbooks Integration:** https://playbooks.com/skills/openclaw/skills/prose

**Integration Points:**
- Works within OpenCode agent framework
- Integrable with various agent harnesses (OpenCode, AMP, CC)
- Supports spinning out Opus 4.5 async subagents
- Enables complex orchestration patterns without rigid baggage
