## Platform Architecture & Core Components

### Four Pillars of Virtuals Ecosystem

1. **Agent Commerce Protocol (ACP)**
   - Standardized framework enabling autonomous agent-to-agent commerce
   - Smart contract-based escrow system for trustless transactions
   - Three-role model: Client (requests/funds), Provider (executes), Evaluator (verifies)
   - Integrated with ERC-8183 standard (co-developed with Ethereum Foundation)
   - Live on Base, Arbitrum, Solana, with expansion to XRP Ledger, BNB Chain, XLayer planned for Q2 2026
   - Processed $600M payment volume through x402 micropayment protocol by November 2025

2. **Butler Agent**
   - User-facing interface agent for accessing agentic supply chain
   - Chat-based entry point for discovering, hiring, delegating tasks
   - Pro mode with research-first, plan-review workflow for complex multi-step tasks
   - Upfront cost estimates and autonomous execution capabilities
   - Recent upgrades: direct task execution, performance-based selection, streamlined wallet flows

3. **Capital Markets (Tokenization & Launchpad)**
   - Agent Tokenization Platform enables capital formation
   - Unicorn Launches as next evolution of token mechanics
   - True co-ownership model for AI agents
   - Genesis protocol-layer primitive for agent token allocation (launched on Base)
   - Ownership earned through contribution, not gated by snipers/cabals
   - Up to $1M/month distributed to agents selling services through ACP (Virtuals Revenue Network)

4. **Robotics/Embodied AI**
   - Eastworld Labs accelerator (launched February 2026) for humanoid robots
   - 500,000 real-world tasks completed through agents controlling physical systems
   - SeeSaw platform partnered with BitRobot Network for training data collection
   - SAM: first on-chain AI agent controlling real-world robots
   - x402 protocol enables agents to autonomously pay for API access, data services, computing
   - Strategic vision: agents must exist in physical world for aGDP to exceed human GDP
